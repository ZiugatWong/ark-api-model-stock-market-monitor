为 windhub.cc 的 Ark API 模型股票提供实时价格监控和数据分析。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击 [Ark API 模型股市监控脚本](https://greasyfork.org/zh-CN/scripts/582321)进入 Greasy Fork 并安装脚本
3. 访问 [windhub.cc](https://windhub.cc) 或 [test-fast.windhub.cc](https://test-fast.windhub.cc)
4. 点击浏览器工具栏的 Tampermonkey 图标，选择相应面板打开

## 分时走势图

**打开方式：** 在"最新价格"面板中点击模型名称

**功能：**
- 显示价格走势曲线
- 今日最高/最低价线标记
- 持仓成本线标记（有持仓时）
- 交易标记点（买入/卖出）
- 支持拖拽移动和八向调整大小
- 数据自动刷新
- ESC 键快速关闭

## 特色功能

- **多面板管理**：所有面板支持独立显示/隐藏和拖拽移动
- **主题切换**：支持日间/夜间模式，覆盖所有 UI 元素
- **价格提醒**：价格突破设定值时自动通知（弹窗/声音/Telegram）
- **数据缓存**：5 分钟缓存机制，减少 API 调用
- **自动清理**：按保留天数自动清理旧价格数据

## 技术栈

- Tampermonkey UserScript
- Lightweight Charts v4.0.1（图表库）
- Web Audio API（声音通知）
- Telegram Bot API（推送通知）
